# 🍱 WhatEat

> 회사 근처 맛집을 동료들과 공유하고, 룰렛·팀투표로 오늘 점심을 빠르게 결정하는 서비스

## 주요 기능
- 🗺️ 내 위치 기반 주변 음식점 지도 (카카오 지도 + 카테고리/가격 필터)
- 🎰 룰렛 — 즐겨찾기 식당 중 랜덤 선택
- 👥 팀 투표방 — 방 코드 공유 후 실시간 투표
- 📖 식사 기록 — 먹은 식당 히스토리 관리
- 🍽️ 식당 상세정보 — 카카오 Place API 메뉴 자동 연동, 상세정보는 네이버 지도 연결

## 기술 스택
- React (Vite) + TailwindCSS
- Supabase (PostgreSQL)
- Kakao Maps JavaScript SDK (지도 표시 + 장소 검색)
- Vercel (배포)
